///////////////////////////// How To Run Project ////////////////////////////

Step 0:
Make sure you have "nodejs" and "mongodb" installed on your system

Step 1:
From your terminal or CMD get into "client" folder and run "npm i" command. 

Step 2:
From your terminal or CMD get into "server" folder and run "npm i" command.

Step 3:
From your terminal or CMD get into "client" folder and run "npm start" command

Step 4:
From your terminal or CMD get into "server" folder and run "npm start" command
